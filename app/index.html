What Powers This Page

GitHub Actions

Every push to the main branch triggers an automated pipeline that tests and deploys this site — no manual steps required.

Terraform

The AWS infrastructure hosting this site is defined in code. The S3 bucket, permissions, and configuration are all version-controlled.

AWS S3 + CloudFront

Static site hosting on Amazon S3 with CloudFront CDN for fast, reliable delivery worldwide.
